# simple-slot-machine
simple slot machine created in python using raylib for graphics
just run the .py file
on your first time running it, the slot machine might not open, just run the .py file again and it should popup, it just means that it needed to install dependencies

CONTROLS
SPACE is roll
UP and DOWN let you change the bet about
R lets you reset your balance
Pressing LEFT SHIFT lets you increase the rate at which you increase or decrease your bet

have fun gambling!
